# Learning- — אוסף כלים קטנים בדפדפן

אוסף כלים עצמאיים שרצים במלואם בדפדפן (HTML/JS, עברית RTL, mobile-first,
עובדים אופליין) ומתפרסמים יחד כאתר GitHub Pages אחד עם דף נחיתה שמקשר לכולם.

**האתר החי:** https://idogol123.github.io/Learning-/

## הכלים

- **portfolio-tracker/** — מעקב תיק השקעות: החזקות, שווי ב-₪/$, רווח/הפסד,
  תמונות מצב וגרף מגמה, לוג הפקדות עם פירוק תשואה אמיתית מול הפקדות, ותכנון
  השפעת הפקדה. הנתונים נשמרים במכשיר בלבד (localStorage). לא ייעוץ השקעות.
- **compound-calculator/** — מחשבון ריבית דריבית: כמה יגדל הכסף עם הפקדה חודשית
  ותשואה שנתית, כולל דמי ניהול, העלאת הפקדה, ותצוגה ריאלית מותאמת-אינפלציה.
  מצב "כמה להפקיד ליעד" (goal-seek), גרף צמיחה, פירוק שנתי ושיתוף לוואטסאפ.
  הנתונים נשמרים במכשיר בלבד. להמחשה בלבד — לא ייעוץ השקעות.
- **guard-duty-scheduler/** — כלי שיבוץ.
- **file-search/** — כלי חיפוש.

## פריסה

`.github/workflows/deploy-pages.yml` מרכיב את האתר בכל דחיפה ל-`main`: מעתיק את
`landing/index.html` לשורש ואת כל תיקיית-כלי לנתיב משלה, ואז מפרסם ל-Pages.

הוספת כלי חדש דורשת עדכון של שלושה דברים יחד: תיקייה חדשה עם `index.html`,
כרטיס מקשר ב-`landing/index.html`, והוספת הנתיב ל-`deploy-pages.yml`
(גם ב-`paths:` של הטריגר וגם בשלב ההרכבה).
