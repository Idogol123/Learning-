# Learning- — אוסף כלים קטנים בדפדפן

אוסף כלים עצמאיים שרצים במלואם בדפדפן (HTML/JS, עברית RTL, mobile-first,
עובדים אופליין) ומתפרסמים יחד כאתר GitHub Pages אחד עם דף נחיתה שמקשר לכולם.
כל כלי מותקן כאפליקציה למסך הבית (PWA) והנתונים נשמרים במכשיר בלבד.

**האתר החי:** https://idogol123.github.io/Learning-/

## הכלים

- **portfolio-tracker/** — מעקב תיק השקעות: החזקות, שווי ב-₪/$, רווח/הפסד,
  תמונות מצב וגרף מגמה, לוג הפקדות עם פירוק תשואה אמיתית מול הפקדות, ותכנון
  השפעת הפקדה. מחירים חיים אופציונליים. הנתונים נשמרים במכשיר בלבד. לא ייעוץ השקעות.
- **compound-calculator/** — מחשבון ריבית דריבית: כמה יגדל הכסף עם הפקדה חודשית
  ותשואה שנתית, כולל דמי ניהול, העלאת הפקדה, ותצוגה ריאלית מותאמת-אינפלציה.
  מצב "כמה להפקיד ליעד" (goal-seek), גרף צמיחה, פירוק שנתי ושיתוף לוואטסאפ.
  להמחשה בלבד — לא ייעוץ השקעות.
- **guard-duty-scheduler/** — שיבוץ שמירות: טבלת שמירות עם שעות שוות בין השומרים,
  איזון לילות וחלוקה הוגנת, ופלט מוכן לשיתוף בוואטסאפ.
- **file-search/** — חיפוש טקסט בתוך קבצים: PDF, Word (docx), Excel (xlsx) וטקסט.
  הקובץ מעובד לגמרי בדפדפן (הספריות מוטמעות בעמוד) — שום קובץ לא עולה לרשת.
  כולל היסטוריית חיפושים, המשך מהקובץ האחרון, ותמיכה ב-PDF מוגן בסיסמה.
- **claude-team/** — אורג-צ'ארט של צוות הכישורים (Skills) של הריפו: 7 מחלקות
  והכישורים שמפעילים אותן, עם קישור אמיתי לכל אחד.
- **sublet-demo/** — דמו UI לאפליקציית שכירות משנה (סאבלט): מסך "אני מחפש" עם
  פיד דירות וסינון (עיר, תאריכים, מחיר, מאומתים) ומסך "אני מפרסם" ליצירת מודעה.
  דמו בלבד עם נתונים מדומים — אין שרת, אין הודעות אמיתיות, הנתונים נשמרים במכשיר.

`landing/` הוא דף הנחיתה שמקשר לכל הכלים ומשמש כשורש האתר.

## פריסה

`.github/workflows/deploy-pages.yml` מרכיב את האתר בכל דחיפה ל-`main`: מעתיק את
`landing/index.html` לשורש ואת כל תיקיית-כלי לנתיב משלה, ואז מפרסם ל-Pages.
שינוי מגיע לאוויר רק כשהוא ב-`main`.

הוספת כלי חדש דורשת עדכון של שלושה דברים יחד: תיקייה חדשה עם `index.html`
(+ נכסי PWA: `manifest.webmanifest`, `sw.js`, ואייקונים), כרטיס מקשר
ב-`landing/index.html`, והוספת הנתיב ל-`deploy-pages.yml` (גם ב-`paths:` של
הטריגר וגם בשלב ההרכבה `cp -r`).

## בדיקות

חלק מהכלים מגיעים עם בדיקות יחידה שרצות ב-Node ללא תלויות:

```bash
node guard-duty-scheduler/test/scheduler.test.mjs   # אינווריאנטים של אלגוריתם השיבוץ
node landing/test/sw-routing.test.mjs               # שה-service worker של הדף לא חוטף נתיבי תת-כלים
```
